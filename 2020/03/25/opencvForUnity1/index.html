OpenCVForUnity笔记1-OpenCv基础函数尝试

发表于 2020-03-25 | 分类于 OpenCvForUnity
尝试使用OpenCvForUnity实现常规图像处理算法

OpenCVForUnity支持OpenCV正式版本的所有操作，C++/Python版本的OpenCV函数在这里都能找到对应的C#版本，函数API可参考官方文档, 遗憾的是文档中没有给出详细的函数说明，有关函数的详细信息需要到OpenCV官方文档中查阅

环境

  • Unity 2019.2.0f1
  • OpenCV for Unity 2.3.7

准备工作

  1. 导入 OpenCV for Unity 2.3.7.unitypackage，unity商店可以搜到
  2. 导入后，点击 Tools / OpenCV for Unity / Set Plugin Import Settings
  3. 移动“OpenCVForUnity/StreamingAssets/”目录到 “Assets/”目录下
  4. 诸如安卓、UWP、IOS等额外的配置细节参阅手册即可

Scalar()函数

Scalar意为标量，其定义为一个由长度为4的数组作为元素构成的结构体，Scalar最多可以存储四个值，没有提供的值默认是0。
