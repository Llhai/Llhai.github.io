Tensorflow Unity尝试

Unity相关

发布于：Mar 25, 2020

环境配置

在Unity上使用Tensorflow需要借助ML-Agents，Unity Machine Learning Agents (ML-Agents) 是一款开源的 Unity 插件， 使得我们得以在游戏环境和模拟环境中训练智能 agents。

  1. Anaconda Tensorflow配置就不写了，本着不给自己找事的原则，使用Py36，tf-1.13.1
  2. 安装 mlagents环境工具，pip install mlagents
  3. 获取ml-agents，在 ml-agents仓库页面直接download所有内容即可。
  4. 进入 解压出的ml-agents目录，安装项目所需文件，有报错基本是依赖版本不符合，卸载原有版本，安装要求版本即可。
  1. 将ml-agents/project/Assets目录下内容拖到Unity中，我在这步时报错了，然后按照官方文档的方法，Select Add package from disk... and navigate into the com.unity.ml-agents folder and select the package.json folder. 然后就可以了
  2. 导入TensorFlowSharp包，这是tensorflowsharp的unity专用包 。该包包含一个tensorflow环境，使你不必依赖系统环境。可以直接跨平台打包。
  3. 后续还有一些诸如在Edit→Player→otherSetting→DefineSymbols中加入 ENABLE_TENSORFLOW等等玄学操作，我也不知道需不需要
