cython-bbox安装报错解决

发表于 2020-05-26 | 分类于 踩坑 |
尝试运行FairMot时遇到缺少cython-bbox的错误，pip 直接安装报错，这里给出错误解决方法

报错信息

这是一个win下很常见的错误，许多linux下的项目搬到win下运行都会遇到这个编译参数错误的问题。

解决方法如下

首先找到报错的是哪个包，我这里遇到的是cython-bbox，在这里下载包的源代码，如图


解压后找到包内的setup.py文件，将extra_compile_args=['-Wno-cpp']改为extra_compile_args = {'gcc': ['/Qstd=c99']}，

最后从本地安装修改后的包即可，具体命令为 pip install -e /解压到的路径/。
